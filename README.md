# AyuFoliaTest

A Minecraft plugin built with [AyuIDE](https://github.com/FORGE096/ayu).

## Platform

- **Server Platform**: Folia
- **Language**: Java

## Building

```bash
./gradlew build
```

## Installation

1. Build the plugin using the command above
2. Copy the generated JAR file to your server's `plugins` folder
3. Restart your server
